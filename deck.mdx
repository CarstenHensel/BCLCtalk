# What is LCIO‚ùì

‚ùó LCIO: Linear Collider I/O  
‚ùó **persistency framework** and **event data model** for linear collider detector studies  
‚ùó intended to be used in both simulation studies and analysis frameworks  
‚ùó allows to easily share results and compare reconstruction algorithms  
‚ùó used by almost all groups involved in linear collider detector studies  

---

# What's ``cvmfs``?

üë®‚Äçüíª The CernVM File System provides a software distribution service.  
üíª developed to deploy software on worldwide-distributed computing  
üìÅ CernVM-FS is implemented as a POSIX read-only file system.   
üåê Files and directories are hosted on standard web servers...  
üìç ...and mounted in the universal namespace ``/cvmfs``.  

https://cernvm.cern.ch/fs/

---
